3 Capabilities Offered by WURFL


Capability Description
form_factor

You can check if a device is a smartphone by testing this capability. See code below:

if (WURFL.is_mobile === true && WURFL.form_factor === "Smartphone"){
//targetSmartPhoneDevices();
}
complete_device_name

You can test an individual iPhone model (Full iPhone model is only available in commercial WURFL.js Basic, Standard, or Pro licenses). See code below:

if (WURFL.complete_device_name === "Apple iPhone XR") {
//targetAppleiPhoneXR();
}
