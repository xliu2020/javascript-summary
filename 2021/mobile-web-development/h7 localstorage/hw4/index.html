Main Characteristics of a PWA

PWA(Progressive Web Application) is a methodolgy to make powerful web applications with the combination of different technologies. It has three main characteristics: reliable,fast and engaging.

3 Main Characteristics of PWA

Reliable - the PWA will load instantly when the app is launched regardless of the network state.

Fast - PWA is fast with no slow-to-respond interfaces even when the application is loaded.

Engaging - user can install PWAs which will live on the user's home screen without the need of any type of app store.
