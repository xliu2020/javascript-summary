Traversy Media

Traversy Media features the online web development and programming tutorials for all of the latest web technologies including Node.js, Angular 2, React.js, PHP, Rails,HTML, CSS and much more.

Traversy Media uploads videos frequently around once a week and the speaker is Brad Traversy. The contents cover a wide range of web technologies as well as some hands-on playful projects. It absolutely deserves a watch.

I have walked through a bunch of its videos and I can always find something new in each of them. Here, I want to introduce two videos I have finished up already.

First Tutorial -- A Practical Guide For Web Development In 2021

This is a practical guide for completely beginners to introduce the roadmap for web developers in 2021 that includes the different technologies to choose from as well as some new trends and some guidance.

First of all, it gives us some insigh into available paths, which are:

  • Frontend Developers
  • Backend Developers
  • Full Stack Developers

Then figure out your career goal. Bard introduced various kinds of tools, covering tools with the most popular trend. According to the path and the goal you choose, pick up the tools that you are in need or are interested in.

Interested in it? Please visit the Website

Second Tutorial -- Create a Website with Video Background

This is a mini project to teach you how to create a beautiful, custom landing page for a Travel website using HTML, CSS and a little Javascript, including embedding the video, customizing the menu and the animation with CSS.

This hands-on tutorial gives me a clear sample on creating a website with a decorated video background and I realize that it is not that hard than I imagined to create a website which follows up the current trend.
