Some websites that might help in web development:

  • W3school: this helps me a lot to understand how does the element works with concrete examples and easy-undestanding explanation. The most important thing is that you can preview the effect of the element we uses, which is good for memorizing these stuff in my opinion.
  • FreeCodeCamp: FreeCodeCamp provides coding exercise for people in different need. No matter you want to develop front-end skills or back-end skills, you can find out what you want in FreeCodeCamp. It helps a lot for learners who start from zero.
  • Flexbox Froggy: this is a game for learning CSS flexbox. The frog is cute and you will have a lot of fun when you learn flexbox. Also, it benefits for memorizing properties with simple defination displayed.
